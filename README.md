# 📊 Super Sales Dashboard with Forecast | Power BI

This project presents an interactive Sales Dashboard built in Power BI to analyze sales performance, profit trends, customer behavior, and future sales forecasting.

## 🚀 Features

* Sales Performance Analysis
* Profit Analysis
* Regional Sales Insights
* Customer Segmentation
* Forecasting & Trend Analysis
* Interactive Filters & Slicers

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
